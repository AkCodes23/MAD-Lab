For the Internal Project, the report must contain (INSEM):
Synopsis
Project Title: A clear, concise name for your application.
Purpose of the App: A brief summary of the app's primary purpose and the problem it solves.
Problem Statement: A detailed description of the issues with existing systems and how your app provides a better solution.
Objectives: Specific, measurable goals the project aims to achieve.
Scope of the Project: The boundaries of the project, defining what features are included in the initial release.
The final report must have (FINAL REPORT):
Abstract 
Introduction
Literature Survey:  At least 10 with a comparison Table
Methodology with Block / Activity/ Database diagram
Results (with screenshots and Inferences)
Limitations
Conclusion and Future Scope
